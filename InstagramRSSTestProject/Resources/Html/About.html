Instagram RSS Test Project

  • In app user can doing login/logout to Instagram with using Instagram API.
  • I added validation for empty/bad login/password.
  • User can loading him rss by signed users & refresh rss by pull down to refresh.
  • All loading data is cached to local storage & will be clean if user doing logout.
  • By selected component user can see a details of selected component.
  • From detail page user can see a page with coments via slide menu.
  • User can set like/dislike for rss-elements.
  • All images is loading on non-main threads.
  • All images are loaded only when needed.
  • Every complex manipulations & handling performed on other threads.
  • This project was developed with using XCode 5 & iOS SDK 7 & ARC.
  • All UI components was created from hardcode & without Storiboards or xib/nib files.
